# Idea Skill

Use the idea workflow to shape rough intent into a Chorus-ready direction:

1. Capture the user goal and relevant context
2. Identify open questions, constraints, and success criteria
3. Separate exploration from implementation work
4. Route proposal-ready work to the proposal workflow

Keep idea notes concise and avoid changing plugin-managed Chorus state directly.
